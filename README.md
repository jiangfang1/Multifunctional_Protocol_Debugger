# Multifunctional_Protocol_Debugger
多功能协议调试器
1、PWM输出
发送“FF 00 55 AA 01 00 00 00 01”
开启PWM输出，不同频率的PWM波输出给到蜂鸣器，产生不同的音频

2、IIC输出
发送“FF 00 55 AA 02 00 00 00 01 03 02 04 25 11 03 01 08 30 00”
从25开始分别指示：年月日，星期，时分秒
将新的时间值通过IIC配置到时间管理芯片，并显示在数码管上

3、USB配置DAC输出4种波形
发送“FF 00 55 AA 03 00 00 00 00”
最后一个值的范围可选00~03
00为正弦波，01为三角波，02为锯齿波，03为方波

4、USB获取ADC值
发送“FF 00 55 AA 04 00 00 00 01”
串口得到一段时间的ADC采样值，对应ADC采样结果

5、USB获取输入信号的频率，占空比，高低电平时间
发送“FF 00 55 AA 05 00 00 00 01”
串口得到4个8bit的值，组成一个32bit的值
比如测量的信号频率为30000000Hz，
串口会收到“01 C9 C3 81”，组合为“01C9C381”，转换成10进制就是30000001Hz，1Hz为测量误差
